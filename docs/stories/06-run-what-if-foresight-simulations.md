# User Story: 6 - Run What-If Foresight Simulations

**As a** hospital director,
**I want** to create and run what-if simulation scenarios across hospital operations,
**so that** I can understand the cascading impact of decisions before implementing them and prevent problems before they happen.

## Acceptance Criteria

*   The director can define a scenario by adjusting variables (e.g., reduce nursing staff by 20, close a ward, change supplier).
*   The simulation engine calculates the cascading impact across all affected modules (staffing, bed allocation, supply chain, finance).
*   Results are presented with clear visualizations showing before-and-after comparisons.
*   Multiple scenarios can be saved, compared side-by-side, and shared with stakeholders.
*   The simulation includes confidence intervals or uncertainty ranges for projected outcomes.

## Notes

*   This is the core value proposition of the platform — described as the "foresight simulation."
*   Specific example given: "What if I fire 20 nurses in December? How does it affect bed allocation? Revenue? Overtime costs?"
*   Hershey referenced prior success with Campari: simulating the shutdown of a Brazil warehouse and its impact on distribution, finance, and client supply.
*   Hershey stated this capability is rare in Southeast Asia for healthcare and is the key market differentiator.
*   Relies on algorithm, trend, and model outputs from the data science layer.
