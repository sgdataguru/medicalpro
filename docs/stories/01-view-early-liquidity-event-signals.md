# User Story: 1 - View Early Liquidity Event Signals

**As a** Relationship Manager,
**I want** to view early liquidity event signals aggregated from multiple data sources on my dashboard,
**so that** I can identify potential opportunities before they become obvious in the market and gain a first-mover advantage.

## Acceptance Criteria

*   Liquidity signals are aggregated from multiple sources including private company databases (e.g., PrivateCircle, Zauba), IPO-related data, corporate actions, margin-related changes, acquisitions, and early exit indicators.
*   Each signal displays the source, type of event, affected individual/company, and estimated timeline (30–90 day triggers).
*   Signals are updated regularly to reflect the latest available data.
*   Each signal is clearly categorized by type (e.g., IPO, VC/PE exit, corporate action, promoter activity).
*   The dashboard presents signals in a clean, uncluttered layout following Kairos Capital branding guidelines.

## Notes

*   The system should aggregate insights from multiple reliable sources rather than relying on a single dataset.
*   For the initial demo, mock data can be used to demonstrate the end-to-end flow.
*   Data sources discussed include PrivateCircle, Zauba, IPO data from exchange and market intelligence sources, promoter information, and public domain disclosures.
