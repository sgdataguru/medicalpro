# User Story: 2 - Filter Prospects by City, Sector, Network, and Cluster

**As a** Relationship Manager,
**I want** to filter the prospect list by city, sector, network, and cluster,
**so that** I can narrow down and focus on the most relevant opportunities within my area of expertise or geography.

## Acceptance Criteria

*   Filter options for city, sector, network, and cluster are clearly presented on the main panel.
*   Selecting one or more filters updates the prospect list in real time.
*   Active filters are visually indicated so the user knows which filters are applied.
*   Filters can be combined (e.g., city + sector) for more precise results.
*   Clearing filters restores the full prospect list.
*   The filter UI remains simple and intuitive per the design requirement.

## Notes

*   The main panel should display these filters alongside top prospects and suggested actions.
*   Design should follow Kairos Capital branding for familiarity and ease of adoption.
