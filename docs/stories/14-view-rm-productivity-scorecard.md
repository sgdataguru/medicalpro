# User Story: 14 - View RM Productivity Scorecard

**As a** Managing Partner,
**I want** to see an RM productivity scorecard showing calls made, conversions achieved, and response times,
**so that** I can identify coaching opportunities, reward top performers, and ensure consistent client engagement.

## Acceptance Criteria

*   Each RM has a scorecard displaying key productivity metrics: calls, conversions, and response times.
*   Metrics can be viewed per RM or as a team-wide summary.
*   The scorecard supports filtering by time period (weekly, monthly, quarterly).
*   Visual indicators (e.g., color coding, trends) highlight underperformance or improvement.
*   The data reflects approximately 30–35 RMs in the organization.

## Notes

*   "RM productivity scorecard (calls, conversions, response times)" was listed as an additional element to incorporate.
*   This supports the managing partner's need for team oversight and performance management.
