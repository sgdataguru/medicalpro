# User Story: 7 - Query Liquidity Insights via AI Chatbot

**As a** Relationship Manager,
**I want** to ask natural-language questions to an AI chatbot that queries the underlying data,
**so that** I can quickly get insights about prospects and liquidity events without navigating multiple screens.

## Acceptance Criteria

*   A live AI chatbot is accessible from within the application.
*   The chatbot can answer questions such as "Which clients may experience liquidity events in the next 30 days?"
*   Responses are based on the actual data in the graph database (not hallucinated).
*   The chatbot supports queries in multiple languages.
*   The chatbot interface is simple and does not disrupt the main workflow.

## Notes

*   This was described as a "nice-to-have if feasible" feature.
*   The chatbot queries the graph database directly to provide answers.
*   Multi-language support was mentioned in the "Page 2 additions" section of the transcript.
